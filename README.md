## Learning Path

### Week 1: Java & Spring Boot Basics
- [ ] Review Java syntax & OOP (if needed): [Java Tutorial for C# Programmers](https://www.tutorialspoint.com/java/java_quick_guide.htm)
- [ ] Read Spring Boot Official Guide: [Spring Boot Getting Started](https://spring.io/guides/gs/spring-boot/)
- [ ] Set up a project with [Spring Initializr](https://start.spring.io/)
- [ ] Install and configure IntelliJ IDEA Community or VS Code with Java extensions

### Week 2: Building REST APIs
- [ ] Learn about Controllers, Services, Repositories
- [ ] Understand Dependency Injection: [Spring Dependency Injection](https://www.baeldung.com/spring-dependency-injection)
- [ ] Build a REST API: [Spring Boot REST API Tutorial](https://spring.io/guides/gs/rest-service/)
- [ ] Write and run tests: [Spring Boot Testing](https://spring.io/guides/gs/testing-web/)

### Week 3: Data Access & Security
- [ ] Use Spring Data JPA: [Spring Data JPA Guide](https://spring.io/guides/gs/accessing-data-jpa/)
- [ ] Connect to a database (H2, MySQL, SQL Server)
- [ ] Add validation: [Spring Boot Validation](https://www.baeldung.com/spring-boot-bean-validation)
- [ ] Implement basic security: [Spring Security Guide](https://spring.io/guides/gs/securing-web/)

### Week 4: Advanced Topics & Best Practices
- [ ] Handle exceptions: [Spring Boot Exception Handling](https://www.baeldung.com/exception-handling-for-rest-with-spring)
- [ ] Use configuration & profiles: [Spring Boot Properties](https://www.baeldung.com/spring-boot-properties)
- [ ] Set up logging: [Spring Boot Logging](https://www.baeldung.com/spring-boot-logging)
- [ ] Monitor with Actuator: [Spring Boot Actuator](https://spring.io/guides/gs/actuator-service/)
- [ ] Learn about deployment (Docker, cloud, etc.)
